# Speech to Verse

A cross-platform church presentation application with real-time speech-to-verse recognition. Type or speak a Bible reference and it instantly displays on a fullscreen output -- like EasyWorship or ProPresenter, but with AI-powered voice control.

## Features

- **Bible Verse Display** -- Search and display any KJV verse on a fullscreen output window
- **Speech Recognition** -- Speak a Bible reference (e.g., "John 3:16") and it automatically appears
- **Voice Commands** -- Say "next verse", "previous verse", or "clear" to control the display hands-free
- **Multi-Verse Ranges** -- Display verse ranges like Psalms 23:1-6 with verse numbers
- **Multi-Monitor** -- Choose which display to use for the fullscreen output
- **Verse Navigation** -- Step forward and backward through verses with click or voice
- **Offline Bible** -- Complete KJV Bible bundled locally (31,100 verses, no internet needed)
- **Cross-Platform** -- Works on Windows and macOS

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
npm run build:bible-db
```

The `build:bible-db` script downloads the KJV Bible and creates a local SQLite database (requires internet once).

### Development

```bash
npm run dev
```

### Production Build

```bash
npm run build
```

## Usage

1. **Manual Search** -- Type a reference like `John 3:16` or `Psalms 23:1-6` in the search bar and press Enter or click Show
2. **Show Display** -- Click "Show Display" to open the fullscreen output window on your selected monitor
3. **Voice Mode** -- Click "Start Listening" to activate the microphone. Speak any Bible reference and it will be recognized and displayed automatically
4. **Navigate** -- Use the Prev/Next buttons or say "next verse" / "last verse" to step through verses
5. **Clear** -- Click Clear or say "clear" to blank the display

## Tech Stack

- **Electron** -- Cross-platform desktop runtime
- **React + TypeScript** -- UI framework
- **Tailwind CSS** -- Styling
- **sql.js** -- SQLite in WebAssembly (offline Bible database)
- **Whisper (via @huggingface/transformers)** -- Local speech recognition (downloads ~40MB model on first use)
- **electron-vite** -- Build tooling

## Voice Commands

| Command | Action |
|---|---|
| "next verse", "next" | Go to the next verse |
| "last verse", "previous verse", "go back" | Go to the previous verse |
| "clear", "blank", "hide" | Clear the display |

## Project Structure

```
src/
  main/           Electron main process (window management, IPC, database)
  preload/        IPC bridge between main and renderer
  renderer/       React UI (control window + display window)
resources/
  bible.db        KJV SQLite database (generated by build script)
scripts/
  build-bible-db  Downloads KJV data and creates the SQLite database
```
